NoorQuran Android / Play Store build steps

Recommended package name:
com.inamaad.noorquran

Live PWA URL:
https://noorquran-eight.vercel.app

Manifest URL:
https://noorquran-eight.vercel.app/manifest.webmanifest

1. Install tools:
npm install -g @bubblewrap/cli

2. Create Android project from PWA manifest:
bubblewrap init --manifest https://noorquran-eight.vercel.app/manifest.webmanifest

When asked, use:
Application ID / package name: com.inamaad.noorquran
App name: NoorQuran
Host: noorquran-eight.vercel.app
Start URL: /
Launcher name: NoorQuran
Display mode: standalone
Orientation: default

3. Build Android App Bundle:
bubblewrap build

4. Digital Asset Links:
After Bubblewrap gives you assetlinks.json or signing certificate fingerprint,
upload the generated assetlinks file to:
public/.well-known/assetlinks.json
Then redeploy the website:
npm run build
npx vercel --prod

5. Upload AAB to Play Console:
Use the .aab file generated by Bubblewrap in Play Console production or testing track.

6. Required Play Console URLs:
Privacy policy: https://noorquran-eight.vercel.app/privacy.html
Support: https://noorquran-eight.vercel.app/support.html
